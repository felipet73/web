Aplicaciones Web

Universidad de los Andes - Departamento de Ingeniería de Sistemas

Descripción de la Asignatura

El curso de Aplicaciones Web está orientado a brindar los conocimientos y habilidades necesarios para diseñar, desarrollar y desplegar aplicaciones web modernas, seguras y escalables. Se abordan tanto los fundamentos teóricos como las herramientas y tecnologías actuales del desarrollo web.

Temas Principales

  • HTML, CSS y JavaScript
  • Frameworks Frontend (React, Angular, Vue)
  • Desarrollo Backend (Node.js, Express, APIs REST)
  • Bases de datos y persistencia
  • Autenticación y seguridad web
  • Despliegue en la nube
  • Buenas prácticas y control de versiones
